# Lienzzo.store

Proyecto web de Lienzzo con interfaz actual, catálogo dinámico, carrusel de imágenes por producto, carrito y compra por WhatsApp.

## Publicación en GitHub + Netlify

1. Sube a GitHub todo el contenido de esta carpeta descomprimida.
2. No subas el archivo ZIP completo; sube los archivos y carpetas internas.
3. Cuando hagas commit en la rama `main`, Netlify publicará automáticamente.

## Estructura principal

- `index.html`: página principal.
- `productos-datos.js`: nombres, precios, descripciones, categorías e imágenes de productos.
- `tooplate-mochi-space.css`: estilos visuales.
- `tooplate-mochi-script.js`: funciones de catálogo, carrusel, carrito y WhatsApp.
- `images/`: logo, banners, iconos y fotos optimizadas de productos.
